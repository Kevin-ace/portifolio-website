Stock Price Dashboard

The Stock Price Dashboard is an interactive web application built using Streamlit and YFinance that allows users to view real-time stock prices, historical price trends, and candlestick charts. Users can select from a list of popular stocks or search for a stock by ticker symbol to view detailed stock information and visualizations.

Sentiment Analysis of Tweets

This web application performs sentiment analysis on recent tweets based on a user-specified keyword. It fetches tweets using the Twitter API and uses VADER (Valence Aware Dictionary and sEntiment Reasoner) for sentiment analysis. The sentiment of each tweet is classified as Positive, Negative, or Neutral, and the results are displayed with a bar chart and a downloadable CSV.

Weather App

A simple weather application that allows users to search for weather information by city name. The app fetches data from the OpenWeatherMap API and displays the current temperature, humidity, and weather description. Users can also toggle between Celsius and Fahrenheit.
